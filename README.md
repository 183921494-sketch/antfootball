# 🐜 蚂蚁足球 — 2026世界杯B2B双终端分析平台

> 为线下彩票店和线上足球预测平台提供专业赛事数据分析服务，采用后置分佣模式（无收益不收费）。

## 📋 项目简介

蚂蚁足球是一个专业的足球赛事数据分析平台，聚焦2026美加墨世界杯，为两类B端合作主体提供数据赋能：

1. **线下实体彩票店** — 提供门店引流、用户维系、销量提升的专业分析内容
2. **线上足球预测平台** — 提供标准化内容输出、付费转化、会员增值的数据支持

核心商业模式：**后置分佣、无收益不收费、零风险合作**。

## ✨ 核心功能

- 📊 **六维分析模型** — 阵容深度/战术体系/关键球员/教练决策/对阵数据/心理意志
- 📈 **MSI评分体系** — 球队综合实力评分（0-10分）
- ⚡ **临场动态修正** — 赛前12h完整报告 + 赛前2h修正版
- 💰 **智能佣金计算** — 根据增量收益自动计算分佣比例
- 📱 **双终端交付** — 彩票店海报/社群推送 + 预测平台API接口

## 🛠️ 技术栈

- **前端**：Next.js 16 + TypeScript + TailwindCSS + Shadcn UI
- **后端**：Next.js API Routes + Supabase PostgreSQL
- **数据库**：Supabase (PostgreSQL + Auth + RLS)
- **部署**：Vercel + Cloudflare CDN
- **AI**：OpenAI SDK + 六维分析模型

## 🚀 快速开始

### 环境要求

- Node.js 22.16.0+
- npm 10.9.2+
- Git

### 安装依赖

```bash
cd antfootball
npm install
```

### 环境变量配置

```bash
# 复制环境变量示例文件
cp .env.example .env.local

# 编辑.env.local，填写你的Supabase凭证
NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
SUPABASE_SERVICE_ROLE_KEY=your_supabase_service_role_key
```

### 启动开发服务器

```bash
npm run dev
```

打开 http://localhost:3000 查看结果。

## 📊 数据库设计

9张核心数据表（已完成设计）：

1. `teams` — 球队基础信息
2. `players` — 球员数据
3. `matches` — 赛程与赛果
4. `odds` — 博彩赔率
5. `analysis_reports` — 分析报告
6. `intelligence` — 情报数据
7. `users` — 用户/合作方
8. `subscriptions` — 订阅与佣金记录
9. `system_config` — 系统配置

## 🤝 合作模式

### 线下彩票店

**分佣规则**（按月度增量纯利润）：
- 1万以内：10%
- 1-3万：8%
- 3万以上：6%

**服务交付**：
- 每日赛事精简解读
- 阶段打包方案（小组赛/淘汰赛）
- 售后支持与答疑

### 线上预测平台

**分佣规则**（按月度付费收益）：
- 5万以内：15%
- 5-15万：12%
- 15万以上：10%
- 独家合作减免2%

**服务交付**：
- 标准化内容输出（每日深度分析长文）
- 付费方案定制（会员专属）
- 数据复盘支撑（赛后复盘+准确率报表）
